# **Examples**

This folder contains examples on how to use dartpollo.

## [**pokemon**](./pokemon)

A simple example using the [Pokémon GraphQL API](https://graphql-pokemon2.vercel.app/). Demonstrates basic schema generation and query execution.

See the [pokemon README](./pokemon/README.md) for setup instructions.

## [**github**](./github)

An example using the [GitHub GraphQL API](https://docs.github.com/en/graphql). Requires a GitHub Personal Access Token. Features custom scalar mappings for GitHub-specific types (`GitObjectID`, `URI`, `DateTime`, etc.).

See the [github README](./github/README.md) for setup instructions.
